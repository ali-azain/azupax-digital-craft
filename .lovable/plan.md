

# Azpax — Premium Digital Agency Website

## Overview
A single-page, dark & bold agency website with smooth scroll navigation, striking typography, and a working contact form powered by Lovable Cloud.

---

## 1. Navigation Bar
- Fixed/sticky top navigation with the Azupax logo on the left
- Smooth-scroll links: Home, About, Services, Portfolio, Testimonials, Contact
- Mobile hamburger menu
- Subtle background blur effect on scroll

## 2. Hero Section
- Full-viewport dark section with bold headline "We Build Digital with Precision"
- Subtitle: "Websites • Branding • Social Media"
- Supporting paragraph text with a "See Our Work" CTA button that scrolls to Portfolio
- Subtle animation on load (fade-in or text reveal)

## 3. About Us Section
- "We Are Azupax" headline with two-column layout (text + visual element)
- Three core value cards with icons: Creativity, Precision, Results
- Closing paragraph about partnering with brands

## 4. Services Section
- Three service cards/blocks with hover effects:
  - **Web Design & Development** — feature list + description
  - **Branding & Graphic Design** — feature list + description
  - **Social Media Management** — feature list + description
- Each with a subtle icon or visual accent

## 5. Portfolio Section
- "Our Work Speaks for Itself" with intro text
- Three project showcase cards (Lunar Café, Zenith Startup, Apex Campaign)
- Each card with project name, key results/stats, and description
- Hover animations for visual polish

## 6. Testimonials Section
- "Trusted by Innovative Brands" headline
- Three testimonial cards with quotes, names, and company names
- Clean card layout with quotation mark styling

## 7. Contact Section
- "Let's Build Something Exceptional" headline
- Working contact form with fields: Name, Email, Message, Service dropdown
- Form validation using Zod
- Backend: Lovable Cloud edge function to receive and store form submissions
- "We respond within 24 hours" note
- "Send Message" CTA button

## 8. Footer
- Logo + slogan: "Azpax — Where Digital Meets Precision"
- Quick links: Home, Services, Portfolio, Contact
- Social media icon links (Instagram, LinkedIn, X, Behance)
- Copyright line

## Design Direction
- **Dark backgrounds** with high contrast white/light text
- Bold, large typography for headings
- Accent color for CTAs and highlights (e.g., electric blue or warm gold — will pick something striking)
- Smooth scroll behavior and subtle scroll-triggered animations
- Fully responsive across all devices
